# Example Artifacts

This folder contains static example outputs generated by the agent.

These files are included so reviewers can inspect the benchmark summary, demo summary, and representative paper-grade reports without running the full system locally.

## Included examples

- `demo_summary.md`
- `benchmark_summary.md`
- `reports/heater_heat_duty_report.md`
- `reports/two_source_blend_report.md`
- `reports/general_blend_report.md`
- `reports/utility_emissions_report.md`
- `reports/utility_repair_report.md`

The reports show generated model results, verification checks, and repair history when applicable.
